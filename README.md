Tower Defence на Arcade  
 Цель создать простую игру, где игрок ставит башни и защищает базу от  врагов

Логика
На экране есть поле и путь, по которому идут враги
Враги двигаются от старта к базе игрока и убивают его, если доходят до конца
Игрок покупает башни за игровое золото и ставит их мышкой на свободные клетки (не на путь)
Башни автоматически стреляют по врагам в своём радиусе
За убитых врагов игрок получает золото и очки

Игра заканчивается
  - поражением, если жизни базы кончились
  - победой, если все волны врагов пройдены

Техническая часть
Язык: Python 3, библиотека arcade.  

Основные классы
  Enemy - враг (здоровье, скорость, движение по пути).  
  Tower - башня (радиус атаки, урон, скорострельность).  
  Bullet - пуля (летит к выбранному врагу и наносит урон).  
  GameWindow - окно игры, создание волн, обработка ввода.

 Состояния экрана
Старт - название игры, краткая инструкция, кнопка/клик Начать.  
Игра - поле, путь, враги, башни, интерфейс (жизни, деньги, счёт, номер волны).  
Финал - сообщение Победа или Поражение, итоговый счёт, предложение начать заново.

По требованиям 
Используются спрайты (Sprite, SpriteList) и базовая анимация движения врагов
Есть два уровня сложности (больше врагов и выше скорость)
Звук смерти врага
Хранение счёта в текстовом файле score.txt
Код разделён на функции/классы, есть комментарии к основным частям.
